This is a standalone HTML test page for Canvas iframe embedding. It uses plain HTML, CSS, and JavaScript. It does not use external libraries, does not contact a server, and does not save or transmit grades.

Test features included: score entry, category weights, lowest quiz dropped, late penalty, dynamic rows, live calculation, reset button, and a simple canvas chart.
